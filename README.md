# BalloonShoot_v2

`BalloonShoot_v2` is a browser game prototype for after-school daycare use.

This repository is the fresh two-camera successor to `BalloonShoot`. The intended setup is:

- front camera for crosshair / aiming
- side camera for trigger / release judgment

The intended interaction is:

- aim with the index finger
- shoot with a thumb-trigger
- pop floating balloons on screen

The repository starts from the original BalloonShoot scaffold so tooling, folder structure, and engineering conventions can be reused while the gameplay/input architecture is rebuilt for dual-camera input.

For Codex remote/cloud preparation notes, see [docs/setup/codex-remote-setup.md](docs/setup/codex-remote-setup.md).
